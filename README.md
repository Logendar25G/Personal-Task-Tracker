# Personal-Task-Tracker
A simple Personal Task Tracker CLI application built using TypeScript. It allows users to add, update, delete, filter, and view tasks directly from the terminal. Tasks are stored in memory, and each task includes status, priority tracking.

# Features

1. Add new Tasks
2. Delete Tasks
3. Update Task Status
4. Filter the tasks by status and priority
5. View All the tasks 
6. Display the tasks in Tabular format


# Tech Stacks

1. TypeScript
2. Node.js
